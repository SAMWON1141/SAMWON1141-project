🚀 PWA 설치 방법

📱 모바일 (Android/iOS)

Android (Chrome, Samsung Internet, Edge)

  1. 웹사이트 접속: 브라우저에서 농장 관리 시스템 접속
  2. 설치 알림 확인: 하단에 "홈 화면에 추가" 배너 표시
  3. 설치 버튼 클릭: "설치" 또는 "홈 화면에 추가" 선택
  4. 확인: "추가" 버튼 눌러 설치 완료
🔔 자동 설치 알림이 표시되지 않는 경우:
  1. 브라우저 메뉴 (⋮) 클릭
  2. "홈 화면에 추가" 또는 "앱 설치" 선택
  3. 앱 이름 확인 후 "추가" 클릭

iOS (Safari)

  1. Safari에서 접속: 농장 관리 시스템 웹사이트 접속
  2. 공유 버튼: 하단 공유 아이콘 (📤) 클릭
  3. 홈 화면에 추가: "홈 화면에 추가" 옵션 선택
  4. 앱 이름 설정: 원하는 이름으로 변경 가능
  5. 추가 완료: "추가" 버튼으로 설치 완료
💡 iOS 팁:
  • Safari 브라우저에서만 PWA 설치 가능
  • Chrome/Firefox에서는 PWA 설치 불가
  • iOS 16.4+ 에서 푸시 알림 지원

💻 데스크톱 (Windows/Mac/Linux)

Chrome, Edge, Opera

  1. 웹사이트 접속: 브라우저에서 농장 관리 시스템 접속
  2. 주소창 아이콘: 주소창 우측 설치 아이콘 (⊕) 클릭
  3. 앱 설치: "설치" 버튼 클릭
  4. 바탕화면 바로가기: 자동으로 바탕화면/시작메뉴에 추가
🖥️ 수동 설치 방법:
  1. 브라우저 메뉴 (⋮) → "앱 설치"
  2. 또는 "더 많은 도구" → "바로가기 만들기"
  3. "창에서 열기" 체크 후 "만들기"

Firefox

  1. 웹사이트 접속: Firefox에서 농장 관리 시스템 접속
  2. 메뉴 열기: 햄버거 메뉴 (≡) 클릭
  3. 페이지 저장: "이 페이지를 앱으로 설치" 선택
  4. 설치 완료: 바탕화면에 앱 아이콘 생성

🎯 PWA 설치 후 사용법

📲 앱처럼 사용하기

  • 홈 화면에서 앱 아이콘 클릭
  • 브라우저 UI 없이 전체 화면으로 실행
  • 네이티브 앱과 동일한 사용 경험
  • 멀티태스킹에서 독립적인 앱으로 표시

🔔 푸시 알림 설정

  1. 알림 권한 요청: 앱 첫 실행 시 알림 권한 허용
  2. 알림 설정: 설정 → 알림 → 원하는 알림 유형 선택
  3. 실시간 알림: 방문자 입장, 시스템 알림 등 자동 수신
📱 지원되는 알림:
  • 새로운 방문자 등록 알림
  • QR 스캔 완료 알림
  • 시스템 상태 알림
  • 긴급 알림 (보안, 에러 등)

📶 오프라인 사용

  • 기본 페이지: 오프라인에서도 접근 가능
  • 방문자 조회: 캐시된 데이터 표시
  • 통계 대시보드: 마지막 동기화 데이터 표시
  • 자동 동기화: 온라인 복구 시 데이터 자동 업데이트

🔧 PWA 관리 및 설정

📱 모바일 PWA 관리

Android

🗑️ 앱 삭제:
  1. 홈 화면에서 앱 아이콘 길게 누르기
  2. "제거" 또는 휴지통 아이콘으로 드래그
  3. 확인 후 삭제 완료
⚙️ 앱 설정:
  1. 설정 → 앱 → "농장 관리 시스템" 선택
  2. 알림, 권한, 저장공간 관리 가능

iOS

🗑️ 앱 삭제:
  1. 홈 화면에서 앱 아이콘 길게 누르기
  2. "앱 제거" 선택
  3. "홈 화면에서 제거" 확인
⚙️ 앱 설정:
  1. 설정 → 일반 → iPhone 저장 공간
  2. "농장 관리 시스템" 선택하여 관리

💻 데스크톱 PWA 관리

Chrome/Edge

🗑️ 앱 제거:
  1. chrome://apps/ 접속
  2. 앱 우클릭 → "Chrome에서 제거"
  3. 또는 시작메뉴에서 우클릭 → "제거"
⚙️ 앱 설정:
  1. 앱 실행 후 메뉴 (⋮) → "농장 관리 시스템 정보"
  2. 권한, 저장공간, 알림 설정 관리

🎉 PWA 장점 및 특징

📈 성능 장점

  • 빠른 로딩: 캐시된 리소스로 즉시 실행
  • 백그라운드 동기화: 오프라인 작업 후 자동 동기화
  • 적은 용량: 기존 앱 대비 90% 적은 저장공간
  • 배터리 절약: 웹 기술 기반으로 효율적 동작

🛡️ 보안 장점

  • HTTPS 필수: 안전한 연결만 지원
  • 샌드박스: 브라우저 보안 정책 적용
  • 악성코드 방지: 웹 표준 기반 안전성
  • 권한 관리: 세밀한 권한 제어 가능

💰 비용 장점

  • 앱스토어 불필요: 배포 비용 제로
  • 즉시 업데이트: 서버 업데이트 시 자동 반영
  • 플랫폼 통합: Android/iOS/Desktop 동일 코드
  • 개발 효율성: 웹 기술로 네이티브 앱 경험

🚨 문제 해결 가이드

설치 버튼이 나타나지 않을 때

1️⃣ 브라우저 확인:
  • Chrome, Edge, Samsung Internet 사용 권장
  • 최신 버전으로 업데이트
2️⃣ 조건 확인:
  • HTTPS 연결 상태 확인
  • 페이지 완전 로딩 대기
  • 잠시 후 다시 시도
3️⃣ 수동 설치:
  • 브라우저 메뉴 → "홈 화면에 추가"
  • 또는 "앱 설치" 옵션 찾기

푸시 알림이 오지 않을 때

1️⃣ 권한 확인:
  • 브라우저/앱 알림 권한 허용 상태
  • 기기 알림 설정 확인
2️⃣ 연결 상태:
  • 인터넷 연결 상태 확인
  • 백그라운드 데이터 허용 설정
3️⃣ 재설정:
  • 앱 제거 후 재설치
  • 알림 권한 재허용

오프라인 기능이 작동하지 않을 때

1️⃣ Service Worker 확인:
  • 브라우저 새로고침 (Ctrl+F5)
  • 캐시 삭제 후 재접속
2️⃣ 저장공간 확인:
  • 기기 저장공간 여유 확인
  • 브라우저 캐시 정리
3️⃣ 재설치:
  • PWA 제거 후 재설치
  • 오프라인 데이터 재다운로드
